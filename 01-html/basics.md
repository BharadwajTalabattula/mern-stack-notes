# Box model:

CSS Box model: it defines how elements are sized positioned and rendered on the webpage; box model components
are content padding border margin

content: the area where text or other content is displayed;
padding: Space between the content and the elements borader;
border: a frame that wraps around the padding and content;
margin: space between the elements border and neighbouring elements;

# Margin --> Border --> padding --> Content 

# box-sizing: 
* box-size css defines how the total height and width of an element are calculated, it allows you to
control wheather padding or boraders are inclued in the elements specifed width and height;

1. content-box (default size)
2. border-box

Content-box: applied for content area --> padding + boarder = not inclued in actual hight and width
border-box: applied for content area --> padding + boarder = inclued in actual hight and width
